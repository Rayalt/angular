Примеры использования пайпов

* DeprecatedDecimalPipe в документации -- устаревшие пайпы, которые скоро будут удалены Работа с числами

number для преобразования чисел

{{ '{{ ' }}e | number:'1.1-5' }}: {{ e | number:'1.1-5' }} (1 число до точки и от 1 до 5 после точки)

{{ '{{ ' }}e | number:'2.1-3' }}: {{ e | number:'2.1-3' }} (2 числа до точки и от 1 до 3 после точки)

Работа со строками

uppercase для верхнего регистра

{{ '{{ ' }}str | uppercase }}: {{ str | uppercase }}

uppercase для нижнего регистра

{{ '{{ ' }}str | lowercase }}: {{ str | lowercase }}

titlecase для каждого слова с заглавной

{{ '{{ ' }}str | titlecase }}: {{ str | titlecase }}

slice обрезает строку

{{ '{{ ' }}str | slice:1:10-1 }}: {{ str | slice:1:10-1 }}

{{ '{{ ' }}str | slice:1 }}: {{ str | slice:1 }}

{{ '{{ ' }}str | slice:1:20 }}: {{ str | slice:1:20 }}

Работа с датами

date форматирует дату из {{ date }}

{{ '{{ ' }}date | date }}: {{ date | date }}

{{ '{{ ' }}date | date:'fullDate' }}: {{ date | date:'fullDate' }}

{{ '{{ ' }}date | date:'longDate' }}: {{ date | date:'longDate' }}

{{ '{{ ' }}date | date:'long' }}: {{ date | date:'long' }}

{{ '{{ ' }}date | date:'y' }}: {{ date | date:'y' }}

{{ '{{ ' }}date | date:'yy' }}: {{ date | date:'yy' }}

{{ '{{ ' }}date | date:'MMMM yy' }}: {{ date | date:'MMMM yy' }}

{{ '{{ ' }}date | date:'M yy' }}: {{ date | date:'M yy' }}

{{ '{{ ' }}date | date:'MM yy' }}: {{ date | date:'MM yy' }}

{{ '{{ ' }}date | date:'MMM, yy' }}: {{ date | date:'MMM, yy' }}

Еще пайпы

{{ '{{ ' }}float }}: {{ float }}

currency добавляет валюту

{{ '{{ ' }}float | currency }}: {{ float | currency }}

percent добавляет знак % и умножает на 100

{{ '{{ ' }}float | percent }}: {{ float | percent }}

{{ '{{ ' }}float | percent:'2.2-3' }}: {{ float | percent:'2.2-3' }}

json не форматирует объект через .toString()

{{ '{{ ' }}obj }}: {{ obj }}

{{ '{{ ' }}obj | json }}: 

{{ obj | json }}

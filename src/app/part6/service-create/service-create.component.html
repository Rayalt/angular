Создание сервисов

Через CLI сервис можно создать командой ng g s --skip-tests component-name

{{ '{{ ' }}appCounterService }}: {{ appCounterService.counter }}

Работает в глобальной области видимости в app.component

{{ '{{ ' }}counterLocalService }}: {{ counterLocalService.counter }}

Работает только в локальной области видимости в service-create.component

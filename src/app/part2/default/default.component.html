Компонент default

Пример вывода стандартного компонента angular, состоящего из:

default.component.html + default.component.scss + default.component.ts

Через CLI компонент можно создать командой ng g c --skip-tests component-name

Флаги: -s for inline css, preventing style file creation --flat to prevent folder creation -t for inline template, preventing html file creation --skip-Tests to prevent .spec file creation
